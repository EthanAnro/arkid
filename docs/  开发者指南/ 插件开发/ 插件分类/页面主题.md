## 使用指南

  租户管理员可以在`租户管理 => 主题设置`页面进行主题的添加和删除，在创建添加主题时，会填入优先级参数，前端读取时会根据优先级的顺序进行响应。如更新或删除主题，需将页面刷新后才会看到前端页面主题的变化。
