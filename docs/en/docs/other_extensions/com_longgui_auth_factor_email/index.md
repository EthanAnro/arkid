# Email 认证因素插件

## 功能介绍

Email认证因素插件，使用邮箱账号来完成用户绑定以及认证

## 使用示例

=== "插件租赁"
    经由左侧菜单栏依次进入【租户管理】->【插件管理】，在插件租赁页面中找到Email认证因素插件卡片，点击租赁
    [![插件租赁](https://s1.ax1x.com/2022/08/10/v1j5g1.png)](https://imgtu.com/i/v1j5g1)
