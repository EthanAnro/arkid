# 账密代填插件

## 功能介绍

通过此插件创建应用后，安装对应的chrome插件，可以实现将用于登录该应用的账号和密码存储在chrome插件里，点击应用图标，可以选择登录的账号，</br>
自动跳转到该应用的登录URL，并自动填入应用登录页账号和密码，实现自动登录，避免每次输入账号密码。</br>
另外，可以在chrome插件里做新增、修改、删除账号密码等管理操作。
